Lynn University Website Redesign

Role

User Interface designer

Tools Used

Sketch 3, and Adobe Photoshop

Challenges

The Creative Director and the Marketing team were looking to redesign the current website structure for Lynn.edu to better reflect their brand message of innovation, student success and diversity.

The current site had many outdated elements that did not reflect the overall message of Lynn, being innovation leaders. As well, many of the site's elements were not cohesive, such as font size, use of dividers, image sizes, etc.

Because the site was not built using the mobile-first approach (designing for smaller screens first, and working your way up), elements such as images and headers were huge when viewed on mobile devices and were not resized properly based on device size.

We were required to utilize the elements that already existed on the website and we were on a time crunch, so I was unable to create wireframes.
